# CSE 340 Course Repository

This repository is dedicated to project work and assignments for the CSE 340 Web Backend Development course. It serves as a central hub for backend application code, server configuration, and database management.

---

## Prerequisites

Before running the application locally, ensure you have the following tools installed:

* **[Node.js](https://nodejs.org/en/download/)** (Includes NPM for package management)
* **[PG Admin](https://www.pgadmin.org/download/)** (For PostgreSQL database management and queries)
* **[Visual Studio Code](https://code.visualstudio.com/)** (Recommended IDE)

---

## Getting Started

### 1. Clone the Repository
Clone this repository to your local machine using VS Code or your terminal:
```bash
git clone [https://github.com/Alderesu/cse340-course-repo.git](https://github.com/Alderesu/cse340-course-repo.git)
```

### 2. Install Dependencies
Navigate into the project directory and install the required Node packages:

```
npm install
```

### 3. Run the Development Server
To start the local backend server, execute:

```
npm start
```

Core Features & Environment
Backend Runtime: Node.js environment handling server-side logic.

Database Management: Structured PostgreSQL integration managed via PG Admin.

Version Control: Managed cleanly with a Node-specific .gitignore to keep local dependencies (node_modules) out of the repository.
